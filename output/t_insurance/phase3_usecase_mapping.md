### 【最優先 Quick Win #1】UC16: 外交員活動支援・提案書自動生成AI

| 項目 | 内容 |
|------|------|
| **概要** | 15,000名の外交員が行う訪問準備・提案書作成・顧客フォローアップをAIが自動化。顧客データと商品知識をRAGで統合した外交員専用AIアシスタント |
| **T保険会社固有の背景** | 外交員1人当たり平均週12-15時間を資料作成・事務処理に費やす。AIで提案書を自動生成することで、顧客との対面時間を大幅拡大し成約率向上に直結 |
| **定量インパクト** | 提案書作成工数▲70%（15,000名×週8h削減換算: **年間7-10億円**相当）。顧客接触時間増加による新規契約増: **3-5億円/年** |
| **実装方法** | 契約DB＋顧客プロファイル＋商品情報をRAGで統合。Claude Sonnet 4.6ベースの外交員向けモバイルAIアシスタント（スマートフォン最適化） |
| **投資額** | 2.0-2.5億円（Year 1） |
| **難易度・規制** | 低（保険募集規制への適合確認は必要。金融庁のモデル提案ガイドライン準拠） |
| **ROI・回収期間** | 4-5ヶ月で投資回収 |
