## このセクションの要点

- 3年累計投資180億円に対しリターン720億円（ROI 300%、NPV 385億円、IRR 82%）。AI導入による営業利益率改善は+1.5〜2.0pt見込み
- リターンの核心は廃棄ロス削減35%（252億円）＋発注業務効率化20%（144億円）。コンビニ業界固有の「食品廃棄コスト構造」がAI投資ROIを製造業より短くする（回収16ヶ月）
- 感度分析の最大リスク変数はFCオーナー導入率（最大影響度±80億円）と廃棄削減実効率（±65億円）。両変数の事前コントロールが投資成否を決定する

## 1. 財務モデルの前提条件と方法論

### 1-1. 主要前提パラメータ

| 前提項目 | 採用値 | 根拠・出典 |
|---|---|---|
| **評価期間** | 5年（Year1〜Year5） | AI投資の長期回収特性。3年を投資集中期、4〜5年を成熟期として区分 |
| **WACC** | 8.0%（保守的採用） | 理論値5.6%に対し、AI投資固有不確実性バッファー+2.4pt加算。詳細はセクション8参照 |
| **税率** | 30% | 実効法人税率（中小企業特例対象外の大手チェーン） |
| **段階的効果発現率** | Year1=30%、Year2=60%、Year3=80% | J-カーブ効果適用。SaaS/Fintech業界AI導入事例の中央値 |
| **FC加盟店配分率** | 廃棄ロス削減効果の本部60%：加盟店40% | 標準フランチャイズ契約（売上総利益分配方式）前提 |
| **廃棄ロス現状** | 売上の4.0%（中央値） | 業界開示データ3〜5%レンジの中点。セブン-イレブン公開データに基づくフェルミ推定 |
| **廃棄ロス削減目標** | AI導入3年後に▲40%削減 | 国内小売AI需要予測導入事例（イオン、ファミマ実証実験）の最頻値 |
| **PB粗利率** | 35% | NB商品平均30%比+5pt。OEM直接調達・自社開発コスト節減分 |
| **店舗数（ベース）** | 15,000店 | 国内大手3チェーン合計推計（7-EI・FamilyMart・Lawson） |
| **1店舗年間売上** | 2.0億円 | 業界平均（日本フランチャイズチェーン協会統計、2024年度） |
| **チェーン全体年間売上** | 3兆円 | 15,000店×2.0億円 |

### 1-2. 方法論注記

本モデルはフェルミ推定ベースの戦略財務モデルであり、以下の三層構造で構築している。

1. **トップダウン試算**（チェーン全体P&Lから逆算）: 廃棄ロス=3兆円×4%=1,200億円/年が削減対象の母数
2. **ボトムアップ試算**（UC別積算）: セクション4のユースケース別ROIテーブルが積算根拠
3. **整合性検証**（①②の差異±5%以内を合格基準）: 両アプローチの整合を確認済み

### 4-3. シナリオ別主要前提の詳細根拠

**アップサイドシナリオ（発生確率25%）の前提**
- FCオーナー導入率90%: 政府のAI補助金活用（IT導入補助金2024拡充）＋競合チェーンの先行導入によるオーナー自発的需要創出
- 廃棄削減▲45%: Year3以降の機械学習モデル高度化（リアルタイム天気・イベントデータ統合）
- PB売上増+5%: AI活用の新商品開発サイクル短縮（6ヶ月→2ヶ月）による市場投入スピード優位

**ダウンサイドシナリオ（発生確率15%）の前提**
- FCオーナー導入率40%: 高齢オーナー層のデジタルリテラシー問題、労組・加盟店組合の反発
- 廃棄削減▲20%: データ品質不足（既存POSシステムの欠損データ25%以上）、モデル精度限界
- API単価+30%: OpenAI/Anthropic等の価格改定、円安進行（155→175円/ドル）

> **[推奨]** ダウンサイドでもROI 120%・NPV 125億円・IRR 38%を確保。元本割れリスクなし。ただし回収期間26ヶ月はYear2後半のCF不足を意味し、**Year1〜Year2の追加ブリッジファイナンス枠（50億円）**を事前確保することを推奨する。

## 7. WACC算定根拠（詳細）

### 7-1. WACC構成要素

| 構成要素 | 採用値 | 算定方法・根拠 |
|---|:---:|---|
| **無リスク利子率（Rf）** | 1.5% | 日本10年国債利回り（2025年度平均実績値） |
| **業界ベータ（β）** | 0.70 | 東証上場コンビニ大手3社の過去5年β平均。安定収益事業のため市場比低め |
| **マーケットリスクプレミアム（MRP）** | 5.0% | 日本株長期超過リターン（Damodaran 2024、東証データ） |
| **株式リスクプレミアム（β×MRP）** | 3.5% | 0.70 × 5.0% |
| **小型株プレミアム** | 0% | 大手チェーン（時価総額1兆円超）につき加算不要 |
| **企業固有AIリスクプレミアム** | 1.5% | AI投資特有の技術・実行リスク。PoC成功率60%、スケール成功率40%から逆算 |
| **株主資本コスト（Ke）** | **6.5%** | 1.5% + 3.5% + 1.5% |
| **負債コスト（税引前）** | 1.0% | 大手コンビニ本部の長期借入金利（格付A〜AA相当） |
| **税引後負債コスト（Kd）** | 0.7% | 1.0% × (1 − 30%) |
| **負債比率（D/（D+E））** | 20% | 保守的資本構成。コンビニ業界平均D/E比率0.25から算出 |
| **株式比率（E/（D+E））** | 80% | |
| **理論WACC** | 5.6% | 0.80 × 6.5% + 0.20 × 0.7% |
| **採用WACC（保守的）** | **8.0%** | 理論値5.6%＋保守バッファー+2.4pt。取締役会ハードルレート8%に整合 |

## 8. 本部：加盟店リターン配分シミュレーション

### 8-1. FC会計特殊性の前提整理

コンビニFCの特徴: 売上総利益（粗利）に対してチャージ率を乗じた額が本部への支払い。廃棄ロス削減はオーナーの仕入れコスト・廃棄処理コスト削減に直結するため、**その効果の帰属先がFC契約条項に依存する**。

| 配分パターン | 本部取り分 | 加盟店取り分 | **加盟店1店年間効果** | FCオーナー導入インセンティブ |
|:---:|:---:|:---:|:---:|:---:|
| **パターンA**（現行ルール準拠） | 60% | 40% | 年間**48万円** | 中（現状コントラクト） |
| **パターンB**（オーナー優遇・推奨） | 40% | 60% | 年間**72万円** | 高（契約期間限定優遇） |
| **パターンC**（廃棄削減全額オーナー帰属） | 0%（AI投資回収のみ） | 100% | 年間**120万円** | 最高（特別条項） |

> **前提**: 全体廃棄削減効果480億円/年（Year3成熟時）÷15,000店=320万円/店。パターンAはその40%×適用発現率で算出。

### 8-2. 配分戦略提言

> **[推奨]** Phase 1（Year1〜Year2）はパターンBを採用し、FCオーナー導入率を加速。「AIでトクした」実感を先に提供することで草の根的な普及を促進する。Year3以降のフル展開フェーズでパターンAに移行し、本部ROI最大化。段階移行はFC契約更改タイミング（通常5年更新）と連動させる。

| フェーズ | 適用パターン | 狙い | 本部インパクト |
|:---:|:---:|---|:---:|
| Year1〜2（PoC・展開期） | **パターンB** | 導入率90%達成 | 本部利益一時圧縮△35億円 |
| Year3〜5（成熟期） | **パターンA** | 本部ROI最大化 | 本部利益+285億円（5年累計） |
| 特別インセンティブ対象店舗 | **パターンC** | 地方・高齢オーナー向け | 先行導入モデル店舗100店限定 |
