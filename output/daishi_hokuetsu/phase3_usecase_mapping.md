> 作成日: 2026年4月14日

---

> **このセクションの要点**
> - 全24ユースケースのうち、Quick Win最優先はUC1（融資審査AI）・UC4（AML/KYC AI）・UC20（渉外担当AIアシスタント）の3件。Year 1から着手し、Month 12までにROI可視化が必須
> - 最高戦略価値はUC12（地域企業DX支援AI）。中小企業向け「無料AI経営診断」から有料コンサルへの移行が新収益源。新潟3万社へのリーチは他行に絶対に作れない独自資産
> - 金融庁規制対応ユースケース（融資AI・AML AI）はシステムリスク管理規制への適合を前提にすること。Phase 0からコンプライアンス部門を巻き込んだ設計が全体スケジュールの鍵

## 3-2. Top 10 ユースケース詳細

### Quick Win Group

#### UC1：中小企業融資審査AI ★★★★★ [最優先]

| 項目 | 内容 |
|-----|-----|
| **概要** | 中小企業融資の信用スコアリングAI。財務データ+法人口座取引データ+外部情報を統合分析 |
| **主機能** | 財務スコアリング自動計算、業種別リスク評価、将来キャッシュフロー予測、担当者向け審査サマリー生成 |
| **ベース技術** | Claude Sonnet 4.6（審査サマリー生成）+ GBM/XGBoostモデル（スコアリング） |
| **Year 1効果** | 審査工数▲50%（年5万件×平均3時間→1.5時間）、承認スピード3営業日→1営業日 |
| **3年累計ROI** | 8億円（工数削減＋承認スピード向上による顧客獲得） |
| **規制分類** | 金融庁システムリスク管理規制対象。モデルの定期的な精度・公平性検証が必要 |
| **着手時期** | Month 5（パイロット：特定業種・支店限定）→ Month 12（全展開） |

#### UC4：AML/KYC AI（疑義取引自動検知） ★★★★★ [最優先]

| 項目 | 内容 |
|-----|-----|
| **概要** | マネーロンダリング・テロ資金供与の疑義取引を自動検知し、調査優先度を自動スコアリング |
| **主機能** | 取引パターン異常検知、顧客プロファイルリスクスコア、疑義取引報告書（STR）自動ドラフト |
| **ベース技術** | 金融特化ML（FICO/NICE Actimize等）+ Claude Sonnet 4.6（STRドラフト生成） |
| **Year 1効果** | 疑義調査工数▲60%、真陽性検知率+35%（誤検知率▲40%） |
| **3年累計ROI** | 6億円（工数削減＋規制罰則リスク低減） |
| **規制分類** | 犯収法・金融庁AMLガイドライン対応必須 |
| **着手時期** | Month 5（既存AMLシステムにAI層を追加）→ Month 10（全面稼働） |

#### UC20：渉外担当AIアシスタント ★★★★★ [最優先]

| 項目 | 内容 |
|-----|-----|
| **概要** | 法人・個人顧客を担当する渉外担当（約800名）向けのAIアシスタント。訪問準備・提案書生成 |
| **主機能** | 顧客財務分析サマリー自動生成、最適商品提案（融資・保険・投信）、訪問記録の音声自動入力、社内規程Q&A |
| **ベース技術** | Claude Sonnet 4.6 + 勘定系データRAG |
| **Year 1効果** | 渉外担当の提案書作成工数▲60%、訪問件数+25%、成約率+10% |
| **3年累計ROI** | 10億円（渉外800名×生産性向上） |
| **着手時期** | Month 5（パイロット30名）→ Month 9（全800名展開） |

### 戦略的長期Group

#### UC12：地域企業DX支援AI ★★★★★ [最重要長期]

| 項目 | 内容 |
|-----|-----|
| **概要** | 新潟県内中小企業3万社向けの「AI経営診断・DX支援プラットフォーム」。銀行の地域貢献＋新収益源 |
| **主機能** | 財務健全性AIレポート、補助金・助成金マッチングAI、後継者問題アドバイスAI、デジタル化支援メニュー提案 |
| **収益モデル** | 基本診断無料（顧客定着）→ 詳細診断・DXコンサル有料（月額3-10万円/社） |

| **着手時期** | Month 14（PoC：100社モニター）→ Month 24（正式リリース） |
