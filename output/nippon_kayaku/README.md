## レポート概要

本レポートは、日本化薬㈱の医薬品事業に特化したMBB（マッキンゼー・ベイン・BCG）水準の生成AI戦略レポートです。ADCリンカー技術という独自競争優位を持つ日本化薬が、生成AIを活用して「ADC技術価値の最大化」と「商業効率の革新」を実現するための包括的な戦略・実行計画を提供します。

## 最優先アクション（今すぐ着手すること）

### 1. 医薬品AI推進室（CoE）設立 — 今四半期中
- 医薬品事業本部長直轄の専任5名チームを2026年Q2中に発足
- 室長候補の選定・内示を2週間以内に開始

### 2. MR支援AIアシスタントのPoC開始 — Month 5まで
- 400名MRの資料作成工数を最大75%削減する最速ROI策
- 6ヶ月で投資回収。9ヶ月で全MR展開完了を目標

### 3. ADCリンカーAIパートナー選定 — Phase 0から並行開始
- Schrödinger等3社と同時PoC契約（各1,000万円・3ヶ月評価）
- ADC技術優位の「分岐点」は2028年。今から着手しないと手遅れ
