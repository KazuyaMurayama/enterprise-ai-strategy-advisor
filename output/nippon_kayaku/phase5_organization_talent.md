# フェーズ5: 組織・人材戦略 — 日本化薬㈱ 医薬品事業

> 作成日: 2026年4月13日

---

> **このセクションの要点**
> - 日本化薬の医薬品事業規模（従業員約2,000名）において、大規模な専任AI組織は不要・不現実。5名のコアCoE（医薬品AI推進室）を司令塔に、既存組織への「AIスポーク」埋め込みによる軽量ハイブリッド型が最適
> - ADC設計AI人材（計算化学×機械学習）は市場で極めて希少。外部採用に依存せず「社内の優秀な研究者にAIスキルを付与する育成転換」が現実的かつコスト効率が高い
> - MRのAI抵抗感が最大のリスク。「AIがMRを助ける」体験を早期に提供し、AI活用を評価指標に組み込むことで能動的受容を促す

---

## 5-1. 組織設計

### 推奨組織モデル：「医薬品AI推進室（CoE）＋スポーク」軽量型

```
                  ┌────────────────────────┐
                  │  医薬品事業本部長        │
                  │  （AI戦略の経営オーナー）│
                  └──────────┬─────────────┘
                             │
               ┌─────────────▼──────────────┐
               │   医薬品AI推進室（CoE）      │
               │   専任5名（Year 1）→ 10名   │
               │                            │
               │  室長（CDO兼務 or 専任）    │
               │  ・AIアーキテクト     1名   │
               │  ・データエンジニア   1名   │
               │  ・Regulatory AI専門 1名   │
               │  ・AI推進PM          1名   │
               └──────────────┬─────────────┘
                              │（支援・標準化）
       ┌──────────────────────┼────────────────────┐
       ▼                      ▼                    ▼
┌────────────┐       ┌──────────────┐     ┌──────────────┐
│R&D AIスポーク│       │Commercial AI │     │Operations AI │
│（兼務2名）  │       │スポーク      │     │スポーク      │
│            │       │（兼務2名）   │     │（兼務1名）   │
│・ADC研究AI  │       │・MR AI推進   │     │・製造AI推進  │
│・臨床試験AI │       │・MA AI推進   │     │              │
│・薬事AI     │       │・CRM AI推進  │     │              │
└────────────┘       └──────────────┘     └──────────────┘
```

> **設計思想**: 日本化薬の規模では、各部門からAI推進担当を「スポーク」として兼務任命（業務時間の30-40%をAI推進に充当）することで、大型投資なく推進力を確保する。CoEは技術・標準・規制対応を一手に担う

## 5-2. CoE 組成計画

### Year 1 医薬品AI推進室（5名体制）

| ポジション | 採用方針 | 必要スキル | 想定年収 |
|----------|:-------:|---------|:-------:|
| **室長（AI推進リード）** | 社内異動（医薬品事業経験者）or 外部採用 | 製薬業界経験10年＋AI戦略策定能力 | 1,200-1,800万円 |
| **AIアーキテクト** | 外部採用（優先） | LLM・クラウドAI設計・MLOps | 900-1,400万円 |

| **Regulatory AI専門家** | 社内育成（薬事部門から選抜） | 薬事規制（PMDA）＋AI/ML理解 | 800-1,100万円 |
| **AI推進PM** | 社内異動（各部門経験者） | プロジェクト管理＋AI基礎理解 | 700-1,000万円 |

## 5-3. 人材育成・リスキリング戦略

### 「日本化薬ファーマAIアカデミー」構想

#### 対象別育成プログラム

| 対象 | 人数 | ゴール | カリキュラム | 期間・コスト |
|-----|:---:|------|------------|:----------:|
| **ADC研究者（研究本部）** | 約80名 | AI創薬ツール活用＋基礎機械学習 | Schrödinger基礎、Python/Pandas、AIツール演習 | 3ヶ月 / 2,000万円 |
| **MR（営業部門）** | 約400名 | MR AIアシスタント完全活用 | MR AI専用研修（8時間）＋ロールプレイ×3回 | 1ヶ月 / 2,500万円 |
| **薬事・QA担当** | 約60名 | AI申請支援ツール活用＋GxP AI理解 | 規制AI専門プログラム（12時間）＋ケーススタディ | 2ヶ月 / 1,000万円 |
| **製造担当（高崎・岡山）** | 約300名 | 製造AIダッシュボード活用 | 製造AI入門（4時間）＋OJT | 2ヶ月 / 1,500万円 |
| **管理職・経営層** | 約80名 | AI経営意思決定リテラシー | AI経営エグゼクティブ研修（1日） | 集中1日 / 500万円 |
| **全社員** | 約2,000名 | AI基礎リテラシー＋倫理 | e-learning（90分）＋ガイドライン | 随時 / 2,000万円 |

**Year 1育成投資合計**: **約9,500万円**（Year 2-3は更新・維持で約4,000万円/年）

#### CoE「ADC AI スペシャリスト」内部育成プログラム

- **対象**: 研究本部の優秀な研究者3-5名（志望者から選抜）
- **期間**: 12ヶ月の集中転換プログラム
- **内容**:
  - 月1-2: AI/機械学習基礎、Python習得
  - 月3-5: ケモインフォマティクス専門（RDKit, Schrödinger API）
  - 月6-8: ADC設計AIプロジェクト実践（CoEメンバーとペア）
  - 月9-12: 独立プロジェクト遂行（ADCリンカーAIパイロット）
- **コスト**: 1名当たり約150-200万円（外部採用比で1/4）
- **目標**: Year 3までに「ADC AI スペシャリスト」5名輩出

## 5-4. チェンジマネジメント

### 日本化薬 医薬品事業のステークホルダー分析

| ステークホルダー | 主な懸念 | 抵抗度 | 具体的対応策 |
|--------------|---------|:-----:|-----------|
| **ADC研究者** | 「AIに仕事を取られる」「AI判断の科学的信頼性」 | ★★★★☆ | 「AIは20個の候補を出すが、選ぶのは研究者」という役割定義を明確化。AI活用で論文・特許数が増えた事例を提示 |
| **MR（最大の抵抗勢力）** | 「AIに管理・監視される」「訪問活動の評価変化への不安」 | ★★★★★ | AI活用MRを「最先端ツールを使いこなすプロ」として社内ブランディング。月1回「AI活用MR表彰」でポジティブ文化 |
| **薬事担当** | 「AI文書の責任所在」「PMDAでの追加質問リスク」 | ★★★☆☆ | PMDA事前相談の結果を薬事担当全員へ共有。AI文書の最終承認は薬事担当が行う体制を明文化 |
| **製造担当** | 「AIに職場を変えられる不安」「バリデーション工数増加」 | ★★★☆☆ | 製造AIダッシュボードを「便利な品質管理ツール」として導入。バリデーション費用はAI予算から拠出 |
| **経営層・取締役会** | 「投資対効果」「規制リスク」 | ★★☆☆☆ | Phase 1完了後に定量成果（ROI）を取締役会でレポート。投資判断の透明性を確保 |

### チェンジ推進カレンダー

| 時期 | アクション | 責任者 |
|-----|---------|:-----:|
| Month 1 | CEO/事業本部長によるAI戦略コミットメント宣言（全社イントラ・全体会議） | 医薬品事業本部長 |
| Month 2-3 | MR向けAIアシスタントβ版体験会（自主参加型、30名パイロット） | CoE + 営業部門 |
| Month 4 | パイロットMRの成果発表（訪問効率・処方増加）を全MR向けに共有 | 営業部門長 |
| Month 5-6 | 全MRへのAIアシスタント展開＋AIスキル研修 | CoE + 営業部門 |
| Month 7 | 「AI活用MR月間MVP」制度開始（月次表彰） | 営業部門 |
| Month 9 | AI活用KPIを全部門の目標に組み込み（人事評価連動） | 医薬品事業本部長 + HR |
| Month 12 | 第1回「日本化薬ファーマAIアワード」開催（年次表彰） | CoE |
