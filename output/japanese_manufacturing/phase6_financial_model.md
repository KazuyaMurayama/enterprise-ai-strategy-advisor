# フェーズ6: 財務モデルとROI試算 — 日本の製造業

> 作成日: 2026年3月31日

---

> **このセクションの要点**
> - 3年間の総投資額は約22億円（技術14億円+人材・組織8億円）、累積リターンは約180-280億円で、累積ROIは720-1,170%
> - ベースケースでの投資回収期間は16ヶ月。ワーストケースでも24ヶ月以内に回収可能であり、投資の下振れリスクは限定的
> - 損益分岐点は累積投資9億円時点（Month 14-16）で到達。Year 1後半から純効果がプラスに転じる

- **データ基盤**: クラウドDWH（BigQuery/Snowflake）+ ETLツール + IoT連携。初年度は構築費、2年目以降は運用・拡張費

- **アプリケーション開発**: 内製エンジニア4-6名×年収1,000万円+開発ツール。外注分はPoCフェーズで多く、Year3は減少
- **AI人材採用**: Year1は5名×平均1,000万円（採用費含む）、Year3は10名規模に拡大

### 損益分岐点の算出

```
累積効果・投資（億円）
     │
 100 │                              ╱ 累積リターン
     │                           ╱
  80 │                        ╱
     │                     ╱
  60 │                  ╱
     │               ╱
  40 │            ╱
     │         ╱           ╱ 累積投資
  20 │      ╱ ─ ─ ─ ─ ╱─ ─ ─ ─ ─ ─
     │   ╱       ╱
   0 ├──●────╱──────────────────────→ 月
     0    6   12  ↑ 18   24   30  36
                  │
            損益分岐点
            （Month 14-16）
```

**損益分岐点**: ベースケースで**Month 14-16**（投資累積約9億円時点）

**推定ロジック**: 
- Month 12時点の累積投資: 約8.2億円、累積リターン: 約7.5億円（NPV）
- Month 16時点の累積投資: 約10.6億円、累積リターン: 約12.5億円（NPV）
- 月次の限界リターンがMonth 7以降に加速し、Month 14-16で累積リターンが累積投資を上回る

> **So What?**
> 
> この投資は「やるべきか否か」の議論ではなく、「いかに速く実行するか」の議論である。ワーストケースでも投資額の2.8倍のリターンが見込まれ、財務的な下振れリスクは限定的。一方、投資しない場合の機会損失は3年間で累積100億円超に達する。経営として合理的な判断は「速やかな投資実行」である。
