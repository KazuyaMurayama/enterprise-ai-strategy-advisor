## 3-3. トップ10ユースケース詳細

### UC-01: 技術文書ナレッジ基盤（RAG）
- **概要**: 過去の設計図面・技術報告書・不具合情報をRAGで検索可能にし、設計者の知識アクセスを革新
- **対象部門**: 設計部門、品質保証部門、製造技術部門
- **期待効果**: ナレッジ検索時間80%短縮（1件30分→6分）、設計ミス20%低減、年間工数削減10,000時間（=約5億円相当）
- **必要な技術スタック**: LLM（Claude/GPT-4クラス）、ベクトルDB（Pinecone/Weaviate）、文書OCR、RAGフレームワーク
- **想定投資額と回収期間**: 初期1.5億円 + 運用3,000万円/年、回収期間6ヶ月
- **リスクと緩和策**: 機密文書の漏洩リスク → オンプレミスまたはVPC内デプロイで対応。データ品質のばらつき → 段階的にデータクレンジング
- **他社事例**: 日立製作所 - Lumada基盤でのナレッジ検索で設計工数25%削減。シーメンス - Industrial Copilotで設計生産性40%向上

---

### UC-02: 品質レポート自動生成・根本原因分析
- **概要**: 品質データから不良トレンド分析レポートを自動生成し、根本原因の仮説を提示
- **対象部門**: 品質保証部門、製造部門
- **期待効果**: レポート作成時間90%削減（8時間→45分）、原因特定時間50%短縮、品質コスト年間10-15億円削減（売上1,000億円企業の場合）
- **必要な技術スタック**: LLM、統計分析ライブラリ、BI連携（Tableau/PowerBI）、SPC（統計的工程管理）データ連携
- **想定投資額と回収期間**: 初期8,000万円 + 運用2,000万円/年、回収期間4ヶ月
- **リスクと緩和策**: AIの分析精度 → 人間によるレビューを必須とするワークフロー。データ統合の複雑さ → 主要ラインから段階導入
- **他社事例**: キーエンス - AI画像検査で不良検出率99.9%達成。ファナック - AIによる加工品質予測で不良率40%低減

### UC-03: 調達交渉支援AI
- **概要**: サプライヤーとの交渉データ・市場価格・コスト構造を分析し、最適交渉戦略と価格目標を提示
- **対象部門**: 調達・購買部門
- **期待効果**: 調達コスト3-5%削減（売上1,000億円企業で製造原価600億円の3%=18億円）、交渉準備時間70%短縮
- **必要な技術スタック**: LLM、原材料市場データAPI、コストモデリングツール、契約書解析AI
- **想定投資額と回収期間**: 初期6,000万円 + 運用1,500万円/年、回収期間2ヶ月
- **リスクと緩和策**: サプライヤー関係への影響 → AI提案はあくまで交渉材料として人間が判断。データの機密性 → アクセス権限の厳格管理
- **他社事例**: トヨタ - AIコスト分析で部品調達コスト2-3%削減。ボッシュ - AI調達プラットフォームで年間€200M以上のコスト最適化

### UC-04: カスタマーサポートAI（技術問合せ対応）
- **概要**: 製品マニュアル・技術仕様書をベースに、顧客からの技術問合せにAIが一次対応
- **対象部門**: カスタマーサポート部門、フィールドサービス部門
- **期待効果**: 問合せ対応の60%自動化、応答時間75%短縮（24時間→6時間平均）、サポートコスト年間3-5億円削減
- **必要な技術スタック**: LLM、RAG、CRM連携（Salesforce等）、チャットUI、エスカレーション管理
- **想定投資額と回収期間**: 初期5,000万円 + 運用1,000万円/年、回収期間3ヶ月
- **リスクと緩和策**: 誤回答による信頼毀損 → 回答の信頼度スコア表示 + 低信頼度は人間にエスカレーション。製品安全に関わる回答 → 安全関連トピックは必ず人間レビュー
- **他社事例**: パナソニック - AIチャットボットで問合せ対応工数40%削減。ABB - AIサポートで初回解決率30%向上

- **必要な技術スタック**: LLM、RAG、CRM/SFAデータ連携、テンプレートエンジン、価格最適化モデル

- リードタイムの短い業界特性を活かし、需要予測AIで在庫回転率30%改善
- 投資額: 約6.3億円（25.3億円×25%）、期待効果: 年間25-40億円

## 3-6. BCGアンビション・マトリクス

ユースケースを「Core（既存事業の最適化）」「Adjacent（隣接領域への拡張）」「Transformational（新規事業創出）」に分類し、投資配分を決定する。

```
                          新規性（高）
                              ↑
                              │
    Transformational          │
    ┌─────────────────┐       │
    │ ・AIサービス事業  │       │
    │ ・マテリアルズ    │       │   投資配分ガイドライン
    │   インフォマティクス│      │   (BCG推奨: 70/20/10)
    │ 【投資: 10%】    │       │
    └─────────────────┘       │
                              │
    Adjacent                  │
    ┌─────────────────┐       │
    │ ・AI設計アシスタント│      │
    │ ・SC自律最適化    │       │
    │ ・営業AI支援     │       │
    │ 【投資: 20%】    │       │
    └─────────────────┘       │
                              │
    Core                      │
    ┌─────────────────┐       │
    │ ・品質検査AI     │       │
    │ ・予知保全       │       │
    │ ・ナレッジ基盤    │       │
    │ ・調達最適化     │       │
    │ ・間接業務自動化  │       │
    │ 【投資: 70%】    │       │
    └─────────────────┘       │
    ──────────────────────────┼──→ 既存事業との距離（遠）
```

| 分類 | ユースケース数 | 投資配分 | 投資額 | 期待リターン | リスク |
|------|:----------:|:------:|:-----:|:---------:|:-----:|
| **Core** | 6件 | 70% | 17.7億円 | 60-80億円/年 | 低 |
| **Adjacent** | 3件 | 20% | 5.1億円 | 20-35億円/年 | 中 |
| **Transformational** | 2件 | 10% | 2.5億円 | 5-15億円/年 | 高 |

> **BCGの知見**: 「70/20/10ルール」は長期的な企業価値最大化に最適。Coreで短期CFを確保しつつ、Transformationalで次の成長エンジンを育てる

## 3-7. 「攻めのAI」ユースケース（新規収益創出）

「守りのAI」（コスト削減・効率化）に加え、製造業の競争優位を根本的に変える「攻めのAI」を3件提案する。

### 攻めUC-1: AI設計自動生成（Generative Design 2.0）

| 項目 | 内容 |
|------|------|
| **概要** | 顧客要件（強度、重量、コスト制約）を入力すると、AIが最適な設計案を複数生成。従来は熟練設計者が2-4週間かかった初期設計を**数時間に短縮** |
| **技術基盤** | 3D生成AI＋物理シミュレーション＋自社設計データでのファインチューニング |
| **投資額** | 3.0億円（3年間） |
| **期待効果** | 設計リードタイム70%短縮、新製品売上20-30億円/年の前倒し獲得 |
| **競争優位** | 自社の設計ノウハウ（暗黙知）がAIに内包され、**模倣困難な差別化資産**となる |
| **リスク** | 設計品質の保証（AIが生成した設計のFMEA）。人間の最終レビュー必須 |

### 攻めUC-2: AIベースの予知保全サービス（BtoB SaaS）

| 項目 | 内容 |
|------|------|
| **概要** | 自社工場で培った予知保全AIを、顧客の設備に対して**サービスとして提供**。製品販売→保守サービス→AIサービスのバリューチェーン延伸 |
| **ビジネスモデル** | 月額サブスクリプション（設備1台あたり10-50万円/月）。ARR目標: Year3で5億円 |
| **投資額** | 2.0億円（3年間） |
| **期待効果** | 新規SaaS事業として年間売上5-10億円。営業利益率40%以上（ソフトウェア事業） |
| **競争優位** | 自社の**OTデータ×製造ノウハウ**がモートとなり、IT企業には模倣困難 |
| **リスク** | 顧客データの取扱い（契約・セキュリティ）、サービス品質保証 |

### 攻めUC-3: サプライチェーン・デジタルツイン

| 項目 | 内容 |
|------|------|
| **概要** | サプライチェーン全体をAIでデジタルツイン化。需給変動、地政学リスク、自然災害などの**What-Ifシミュレーション**をリアルタイムで実行 |
| **技術基盤** | グラフニューラルネットワーク＋大規模最適化＋リアルタイムIoTデータ |
| **投資額** | 3.5億円（3年間） |
| **期待効果** | 在庫削減15-20%（5-8億円/年）、リスク事象発生時の対応リードタイム80%短縮 |
| **競争優位** | サプライヤーとのデータ連携がネットワーク効果を生み、**後発参入障壁**となる |
| **リスク** | サプライヤーのデータ共有への抵抗。段階的アプローチ（主要10社から開始）が必要 |

---

> **So What?**
> 
> Top10ユースケースの合計初期投資15.6億円に対し、年間効果は71-124億円と、極めて高いROIが見込まれる。特にQuick Win象限の5案件（投資合計3.8億円、年間効果39-48億円）は、技術的成熟度が高く6ヶ月以内の成果可視化が可能であり、経営層の意思決定を促進する「成功体験の早期創出」に最適である。
>
> さらに、セグメント別戦略により投資の最適配分が可能となり、BCGアンビション・マトリクスの「70/20/10ルール」でCore（短期CF確保）とTransformational（次の成長エンジン）のバランスを確保する。「攻めのAI」3件は、製造業の競争構造を根本的に変える可能性を持ち、特にAI設計自動生成と予知保全SaaSは**自社の暗黙知をAIに内包する**点で模倣困難な差別化となる。
