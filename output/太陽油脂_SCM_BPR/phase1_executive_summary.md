# Phase 1: エグゼクティブサマリー — 太陽油脂株式会社 SCM-BPR

> 作成日: 2026年4月21日

---

> **このフェーズの要点（役員会向け3行）**
> - 業務改革部SCM領域26名のうち▲7名（▲27%）削減は、判断業務を客観的に定義した上での工数積み上げにより**技術的・運用的に実現可能**
> - 5月20日時点での決議事項は「人員削減承認」ではなく「**判断業務15項目の定義承認**」。削減の根拠を先に固めることが最大のリスク回避
> - 年間4,900万円の人件費削減に対し初期投資3,000〜5,000万円、ペイバック約1年。3年累計ネット効果は**約1億円**

## 1-1. 結論：3つのKey Message

### Key Message 1: 削減可能工数は年間約11,500時間（7名相当）

業務改革部26名の年間総工数（約43,680時間）のうち、AI/RPA化・外注化・業務廃止により削減可能な工数は**約11,500時間**。これは7名の年間標準稼働時間（1,680h/名×7=11,760h）にほぼ合致する。

### Key Message 2: 削減の82%はAI/RPA+業務廃止で代替可能

| 代替手段 | 削減工数 | 割合 |
|---------|:-------:|:---:|
| RPA完全自動化 | 3,200h | 28% |
| 生成AI+RPA（Human-in-Loop） | 5,100h | 44% |
| 一部外注化 | 1,200h | 10% |
| 業務廃止・標準化 | 2,000h | 17% |
| **合計** | **11,500h** | **100%** |

### Key Message 3: 判断業務として保護すべき業務は全体の約30%

AI・RPA化しない（すべき）「判断業務」は業務全体の約30%（チーム別：生産計画35%・物流25%・受注10%・業務総括60%）。これを先に定義することで、「AIで全員削減」という誤解を組織的に解消できる。

## 1-5. 論理ピラミッド：なぜ▲7名が妥当か

```
【結論】業務改革部26名 → 19名（▲7名）は妥当
        │
        ├── 【根拠1】削減可能工数が7名相当（11,500h）存在する
        │         └── 120業務の3軸スコアリングで客観的に特定
        │
        ├── 【根拠2】代替技術が実装可能
        │         ├── RPA/OCR: 受注入力・実績集計（即時実装可能）
        │         ├── 生成AI+最適化: MC品目計画支援・配車最適化
        │         └── 外注: 物流の一部定型業務
        │
        └── 【根拠3】判断業務は保護した上での削減
                  └── 15項目の判断業務を定義・保護した残余が削減対象
```

## 1-6. 本プロジェクトのReframing（最重要前提）

> **本プロジェクトは「人員削減プロジェクト」ではなく「業務再定義プロジェクト」である**

このReframingが成功の鍵。理由：

| 旧フレーム（危険） | 新フレーム（本プロジェクト） |
|---|---|
| 削減数（▲7名）から逆算して業務を切る | 業務を定義してから削減数が導き出される |
| AI化できるものをできるだけ多く | 人間が必須な業務を先に守る |
| 現場は削減される側 | 現場は業務定義の主体者 |
| 結論先行で役員会承認 | 根拠先行で役員会承認 |

> **エグゼクティブサマリー総括**: 太陽油脂業務改革部SCM領域の▲7名（26名→19名）は、「判断業務の客観的定義」→「AI/RPA適用可否判定」→「工数ボトムアップ積み上げ」の順序で論理的に導き出される。5月20日の最重要成果物は人員削減計画ではなく「判断業務定義文書」であり、これを役員会で承認することが、9月20日最終承認への確実な論理基盤となる。
