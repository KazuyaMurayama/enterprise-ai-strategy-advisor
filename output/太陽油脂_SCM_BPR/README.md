## 設計思想：なぜ通常のAI戦略レポートではないのか

> **本レポートは「AI戦略レポート」ではなく「BPR実装設計書」である**

| 論点 | 一般的AI戦略レポート | 本レポートのアプローチ |
|---|---|---|
| 意思決定の性質 | 投資判断・全社戦略 | 人員計画・組織設計の根拠提示 |
| 時間軸 | 3-5年ロードマップ | **5月20日役員会までの4週間設計** |
| 成果物の質 | ビジョン・機会マッピング | **監査可能な判断業務定義書** |
| 数値根拠 | 業界ベンチマーク流用 | **自社工数データ積み上げ** |
| 最大リスク | 技術選定ミス | **人員削減の論理破綻による労務リスク** |

**最上位目的:** 「削減数ありき」の危険な思考順序を逆転させ、「業務定義 → 自動化可否 → 必要工数 → 必要人員」のボトムアップで役員会承認に耐える論理を構築する。

## 判断業務分類サマリー（Phase 3 核心）

```
3軸スコアリング（各0-3点、合計9点満点）
  軸A: 情報完全性（構造化データで完結するか）
  軸B: 判断再現性（同じ入力で同じ判断が出せるか）
  軸C: 例外発生頻度（想定外ケースの頻度）

分類基準:
  0-2点 → 実行業務（AI/RPA化可能）
  3-4点 → 準判断業務（部分AI化＋人間レビュー）
  5-6点 → 判断業務（人間主体、AI支援のみ）
  7-9点 → 高度判断業務（人間必須、AI代替不可）
```
