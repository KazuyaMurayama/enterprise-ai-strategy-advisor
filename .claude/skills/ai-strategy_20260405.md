# 生成AI戦略レポート自動生成

ユーザーが指定した企業名または業界名に対して、マッキンゼー・ベイン・BCGクラスの生成AI戦略レポートを全8フェーズで自動生成します。

## 実行手順

引数として企業名または業界名を受け取ります: `$ARGUMENTS`

### ステップ1: 情報収集
WebSearchとWebFetchを使用して、対象企業/業界に関する以下の情報を収集:
- 企業の基本情報（売上高、従業員数、事業構成、主要KPI）
- 業界のAI導入状況と最新トレンド
- 主要競合のAI戦略と投資状況
- 関連する規制・法的環境
- 市場規模と成長予測

### ステップ2: フェーズ別レポート生成
以下の8フェーズを順番に実行し、各フェーズの成果物を個別ファイルとして保存:

**フェーズ1: エグゼクティブサマリー**
- 結論（3行）、想定インパクト、推奨投資規模、最優先アクション上位3つ
- 着手しない場合のリスク、意思決定期限

**フェーズ2: 業界/企業の現状分析**
- PESTEL + 生成AIトレンド分析
- Porter's Five Forces + AI変数分析
- 競合ベンチマーク（3-5社、表形式）

**フェーズ3: バリューチェーンAI機会マッピング**
- バリューチェーン各工程のAIユースケース特定
- プライオリティマトリクス（2x2）
- トップ10ユースケース詳細

**フェーズ4: 技術アーキテクチャ戦略**
- Build vs Buy vs Partner分析
- 基盤モデル選定ガイドライン
- データ戦略

**フェーズ5: 組織・人材戦略**
- 組織設計（CoE型/分散型/ハイブリッド型）
- 人材スキルマトリクス
- チェンジマネジメント計画

**フェーズ6: 財務モデルとROI試算**
- 3年間の投資計画と財務サマリー表
- 感度分析（3シナリオ）
- 損益分岐点

**フェーズ7: 実行ロードマップ**
- Phase 0-4（36ヶ月）の詳細計画
- 各フェーズのGo/No-Go判断基準

**フェーズ8: リスク管理**
- リスク一覧と対策表
- AI倫理・ガバナンスフレームワーク

### ステップ3: 統合と保存
- 全フェーズを統合した最終レポートを作成
- `output/{企業名or業界名}/` 配下に全ファイルを保存
- 目次ファイル（README.md）を作成
- GitHubにコミット・プッシュ
