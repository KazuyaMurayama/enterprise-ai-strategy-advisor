# 財務モデル・ROI試算

対象: `$ARGUMENTS`

## 実行内容

### 6-1. 投資計画
年度別投資額の内訳を算出:
- 技術投資（インフラ、ライセンス、開発費）
- 人材投資（採用、育成、外部コンサルタント）
- 組織投資（チェンジマネジメント、教育）
- 外部委託費
- 3年間の総投資額

### 6-2. リターン試算
前提条件を明示した上で以下を算出:
- 売上向上効果（新規収益、クロスセル、アップセル）
- コスト削減効果（自動化、効率化、エラー削減）
- 生産性向上効果（時間短縮、品質向上）
- 3年間のROI推計

### 6-3. 財務サマリー表

```
項目            | Year1      | Year2      | Year3      | 累計
---------------+-----------+-----------+-----------+-----------
投資額          |           |           |           |
売上向上効果    |           |           |           |
コスト削減効果  |           |           |           |
純効果          |           |           |           |
累積ROI         |           |           |           |
```

### 6-4. 感度分析
- ベストケース / ベースケース / ワーストケース の3シナリオ
- 主要変数の感度テーブル:
  - 導入速度（計画比 80%/100%/120%）
  - 効果発現率（期待値の 60%/80%/100%）
  - コスト変動（予算比 90%/100%/120%）
- 損益分岐点の算出（月数）

## 入力パラメータ
引数として以下を受け付ける:
- 企業名または業界名
- 従業員数（任意）
- 売上高（任意）
- 業界平均利益率（任意）

## 出力形式
- `output/{対象名}/phase6_financial_model.md` に保存
- すべての数値に前提条件とロジックを明記
