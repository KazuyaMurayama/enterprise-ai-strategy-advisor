# 業界/企業分析スキル

対象: `$ARGUMENTS`

## 実行内容

### 2-1. マクロ環境分析（PESTEL + 生成AIトレンド）

以下の6つの観点から分析を実施:

- **政治/規制**: AI規制動向（EU AI Act、日本AI戦略等）の当該業界への影響
- **経済**: 生成AI市場規模の推移と予測（具体的数値を含める）
- **社会**: 労働力・スキルギャップ、消費者のAI受容度
- **技術**: 基盤モデルの進化速度、マルチモーダル化、エージェント化の影響
- **環境**: AI計算コストのエネルギー問題、サステナビリティとの整合性
- **法務**: 著作権、個人情報保護、AI生成物の責任所在

### 2-2. 業界固有の競争環境分析（Porter's Five Forces + AI変数）

各Forcesに対して生成AIがどう力学を変えるかを分析:
- 新規参入の脅威 + AI変数
- 代替品の脅威 + AI変数
- 買い手の交渉力 + AI変数
- 売り手の交渉力 + AI変数
- 既存競合間の競争 + AI変数

業界のAI導入成熟度を5段階で評価:
- レベル1: 未着手 / レベル2: PoC段階 / レベル3: 部分導入 / レベル4: 全社展開 / レベル5: AI-Native変革

### 2-3. 競合ベンチマーク

- 主要競合3-5社のAI戦略を調査・比較
- 各社の投資額、導入事例、成果を表形式で整理
- 競合対比でのポジショニングマップ（テキスト形式）

## 出力形式
- Markdownファイルとして `output/{対象名}/phase2_industry_analysis.md` に保存
- WebSearchで最新データを取得して数値を含める
- すべての数値に出典または推定根拠を付記
